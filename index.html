A for Animation

ease-in-out

Mon petit cube rouge présente une animation qui utilise plusieurs propriétés donc le transform: translate et le transform: rotate et les keyframes. Voir le code plus bas.

.box{ margin: bottom 40px; width:100px; height:100px; background-color: #e02020; animation: example 2s ease-in-out 1s infinite alternate both; } @keyframes example{ from{ transform: translate(-200px, -100px) rotate(0); } to{ transform: translate(400px, 200px) rotate(360deg); } }

  1. border-image-outset: Specifies the amount by which the border image area extends beyond the border box

  5. border-image-width:permet de spécifier la largeur de la bordure.

J for justify

justify-content

Cette propriété sans laquelle notre vie ne serait définitivement pas la même, permet d'aligner les éléments sur l'axe principal. Cette propriété est utilisé avec un display: flex. En voici un exemple:

item1

                    .justify_exemple{
                        display: flex;
                        flex-direction: row;
                        background-color: var(--secondary-color);
                        width: fit-content;
                        gap:1rem;
                        padding: 1rem;
                    }
                
                    .justify_content{
                        justify-content: space-evenly;
                        align-content: center;
                        background-color: var(--light-pink);
                        height: 50px;
                        width: 50px;
                    }
  1. flex-start: aligner au début
  2. flex-end: aligner à la fin
  3. Center: aligner au centre
  4. space-between: met un espace entre les éléments
  5. space-around: met un espace autour des éléments
  6. space-evenly: qui distribue l'espace également

Z-index

Dans l'exemple plus haut, vous verrez aussi l'utilisation de la propriété transform:translate()
